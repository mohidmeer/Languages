# Data Types

## NUMBERS

```javascript lineons
    let age = 30
    let price = 19.99
    let temperature = -10
```

## STRINGS

```javascript lineons
    let name = "John Doe"
    let message = 'Hello, how are you?'
    let address = `123 Main Street`
```

## BOOLEAN

```javascript lineons
    let isStudent = true
    let hasCar = false
```

## UNDEFINED

```javascript lineons
    let undefinedVariable =undefined
```

## NULL

```javascript lineons
    let nullVariable = null
```

## ARRAY

Can hold any type of **Primitive** or **Complex Data**

```javascript lineons
    let colors = ["red", "green", "blue", "yellow"];
    let numbers = [1, 2, 3, 4, 5];
```

## OBJECTS

Objects can also have methods inside it which are simply functions that we can call on that object

```javascript lineons
    let person = {
    name: "Alice",
    age: 25,
    isEmployed: true,
    address: {
    street: "123 Main Street",
    city: "Anytown",
    country: "USA"
     }
    }
```
